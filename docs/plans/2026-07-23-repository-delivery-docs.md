# Repository Delivery Documentation Plan

**Goal:** Make the public Loop Craft repository understandable and runnable by a new contributor without overstating unsupported features or hiding the pending license decision.

**Scope:** Root README, repository structure/navigation, current capability boundary, install/use/build instructions, Evidence/Artifact separation, reference attribution, and current governance links. License files and CI remain separate decisions until authorized.

**Acceptance:** A new visitor can identify what Loop Craft is, what is implemented, how to install/use/build/verify it, what is out of scope, where evidence lives, and why the public repository currently has no redistribution license.
